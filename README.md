# WeatherBuddy 🌤️
 
A React weather application that fetches real-time weather data using an external weather API.
 
## Features
- Search weather by city name
- Display current temperature, conditions, and weather details
- Responsive layout
- API integration with weather service
  
## Tech Stack
- React (Create React App)
- CSS
- Weather API (OpenWeatherMap or similar)

## How to run
1. Clone the repo
2. Run `npm install`
3. Create a `.env` file with your API key (do NOT commit this file)
4. Run `npm start`

## Note
This was built in September 2024 as a learning project — one of the first React apps built with API integration. Code was written block by block with AI assistance as part of a structured learning approach.
